getItems(element)

 

Return a dictionary of the items of an element, with index 0 of the itemslist as key and index 1 of the itemslist as value. This is an element as returned by the pymzml parsing of an mzml file

Parameters:
  • element (Element from cElementTree) - Element
Returns: dictionary
The items of element

Example:

Print the items of all the elements in a featureXML file

>>> from xml.etree import cElementTree
>>> elementFile = open('exampleFeatureXMLfile.featureXML')
>>> for event, element in cElementTree.iterparse(elementFile):
...    print getItems(element)

{'type': 'string', 'name': 'parameter: in', 'value': 'C:/Users/DMAMAR~1/AppData/Local/Temp/TOPPView_2011-04-05_164711_LS23428_1372_1_in'}
>>> elementFile.close()

memoryUsage()

 

Memory usage of the current process in kilobytes. Gets the VmRSS and VmPeak from the status file (as peak and rss).

  • VmPeak is the maximum virtual memory space used by the process, in kB (1024 bytes).
  • VmRSS is the amount of memory that have been mapped into the process' address space, or its resident set size

Only works on linux because it looks in a linux specific file (/proc/self/status)

Returns: dictionary
The maximum virtual memory space used by the process(peak) and the amount of memory that have been mapped into the process' address space(rss)

Example:

Print the peak memory usage and the resident set size (probably only works on linux):

>>> result = memoryUsage() 
>>> print 'The peak memory usage is:',result['peak'],'and the resident set size is:',result['rss']
The peak memory usage is: ... and the resident set size is: ...

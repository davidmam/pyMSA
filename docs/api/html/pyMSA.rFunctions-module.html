index(matrix, columnName)

source code 

Get the column index number columnName in matrix.

The index returned is a 'python' index, meaning it starts from 0. R indexing starts at 1, so if you want to use this with an R function add 1 to the return value.

Parameters:
  • matrix (rpy2.robjects.vectors.DataFrame) - Matrix or DataFrame with named columns.
  • columnName (string) - Name of one of the columns in matrix
Returns: number
Index of the given column name in the given matrix
Raises:
  • KeyError - columnName does not exist in matrix
  • TypeError - matrix is not of type rpy2.robjects.vectors.DataFrame

Attention: Returns a 'python' index, starting at 0. R indexes start at 1.

Examples:

Printing the index of the column 'test1' in exampleFrame:

>>> import rpy2.robjects as R
>>> dict = {'test1': R.IntVector((12,12,15)), 'test2': R.IntVector((32,4,12)), 'test3': R.IntVector((3,12,26))} 
>>> exampleFrame = R.DataFrame(dict)     # example dataframe. Important to note: because of dictionaries ordering mechanism, in this example the order of columns is test1, test3, test2. The ordering does not affect index().
>>> print index(exampleFrame, 'test1')   
0

Using the column index to extract the values of column 'test1' from testFrame with python's lists bracket syntax:

>>> import rpy2.robjects as R
>>> dict = {'test1': R.IntVector((12,12,15)), 'test2': R.IntVector((32,4,12)), 'test3': R.IntVector((3,12,26))}       
>>> exampleFrame = R.DataFrame(dict)     # example dataframe.   
>>> print testFrame[index(testFrame, 'test2')]   # get the values of column 'test2'
[1] 32  4 12

Using the column index to extract the values of column 'test1' from testFrame with R's .rx() function. Note that +1 is added to the index because R's indexing starts at 1 instead of 0:

>>> import rpy2.robjects as R
>>> dict = {'test1': R.IntVector((12,12,15)), 'test2': R.IntVector((32,4,12)), 'test3': R.IntVector((3,12,26))}    
>>> exampleFrame = R.DataFrame(dict)    # example dataframe
>>> print testFrame.rx(True, index(testFrame, 'test2')+1)  # note the +1, because testFrame.rx() is an R function, so the indexing starts at 1
[1] 32  4 12

getRowsWithUniqColumn(matrix, columnName)

source code 

Get all the rows from matrix with unique value in the column columnName.

Parameters:
  • matrix (rpy2.robjects.vectors.DataFrame) - The matrix that you want a subset of
  • columnName (string) - The name or index of the column you only want the unique values of
Returns: rpy2.robjects.vectors.DataFrame
A subset of matrix with only unique values in column.
Raises:
  • TypeError - column is not of type str

    Example:

    Printing a sub dataframe where the values in column 1 have to be unique:

    >>> import rpy2.robjects as R
>>> dict = {'test1': R.IntVector((12,12,15)), 'test2': R.IntVector((32,4,12)), 'test3': R.IntVector((3,12,26))}       # note that test1 has 12 in row 1 and row 2 
>>> exampleFrame = R.DataFrame(dict)    
>>> print getRowsWithUniqColumn(testFrame, 'test1')  # now only two of the three rows remain, because test1 had 12 twice in the row
  test1 test3 test2
1    12     3    32
3    15    26    12

getColumns(matrix, *args)

source code 

Make a subset of a matrix or dataframe containing only the columns with column names given to *args.

Parameters:
  • matrix (rpy2.robjects.vectors.DataFrame) - The matrix that you want the subset of
  • args (string) - One or more column names that you want in the subset
Returns: rpy2.robjects.vectors.DataFrame
Subset of matrix only containing columns given to *args
Raises:
  • InputError - No arguments given to *args
  • TypeError - matrix is not of type rpy2.robjects.vectors.Matrix or rpy2.robjects.DataFrame
  • TypeError - Names passed to *args are not of type str

    Example:

    Print a sub dataframe with only 2 of the 3 original columns:

    >>> import rpy2.robjects as R
>>> dict = {'test1': R.IntVector((12,12,15)), 'test2': R.IntVector((32,4,12)), 'test3': R.IntVector((3,12,26))}       # note that test1 has 12 in row 1 and row 2 
>>> testFrame = R.DataFrame(dict)    
>>> print getColumns(testFrame, 'test1', 'test3')
  test1 test3
1    12     3
2    12    12
3    15    26

takeLog(x, base)

source code 

Take the logarithm of an R type object. The R type is tested on a vector, see R documentation about ?log what other types can be used.

Parameters:
  • x (R variable that works with log() e.g. IntVector. (see R documentation ?log)) - Object that you want to take the log of.
  • base (Number) - base for the logarithm
Returns: Same as x
The logarithm of all the values in x.

Example:

Print the logarithm of a vector:

>>> import rpy2.robjects as R
>>> vector = R.IntVector((59843, 34982, 12425, 90534, 34532, 54642, 1239, 43534))
>>> print takeLog(vector, 10)
[1] 4.777013 4.543845 4.094296 4.956812 4.538222 4.737527 3.093071 4.638829
Raises:
  • ValueError - base not a positive number

readCsvFile(csvFilePath, sep='\t', head=True, na='N/A', **kwargs)

source code 

Read a csv file and return the content. Returns a dataframe of the csv file data

Parameters:
  • csvFilePath (string) - The path to the csv file
  • sep (string) - The character to separate the csv file on (default = ' ')
  • head (bool) - Boolean if the file to read in has headers or not (default: True)
  • na (string) - What do the non available values look like (default = N/A)
  • kwargs - Any additional options for to R.r['read.csv'] (read R's ?read.csv documentation to see what arguments are possible)
Returns: rpy2.robjects.vectors.DataFrame
Dataframe representation of the csv file

Examples:

reading a csv file separated by tabs, with headers and N/A as not available (the default settings):

>>> csvDataFrame = readCsvFile('exampleFile.csv')

reading a csv file separated by spaces, no headers, not available as NA, and escapes are allowed:

>>> csvDataFrame2 = readCsvFile('exampleFile2.csv', sep=' ', head=False, na='NA', allowEscapes='True')

Fills the given vector with the 'not available' R type (either NA_Logical, NA_Real, NA_Integer, NA_Character or NA_Complex)

Parameters:
  • vectorList (list) - List to which 'not available' types will be appended.
  • amount (number) - The amount of 'not availables' to add to vector
  • rType (The type of 'not available' to use (either NA_Logical, NA_Real, NA_Integer, NA_Character or NA_Complex))
Returns: list

Raises:
  • TypeError - rType did not contain allowed value
  • TypeError - vectorList is not of type list

    Example:

    Add 2 Na_Integers to a list

    >>> print rFunctions.fillNA([1,2,3], 2, 'NA_Integer')
[1,2,3, R.NA_Integer, R.NA_Integer]

mz_rt_plot()

 

Example of how to plot the m/z values against the retention time by selecting features from a window of m/z values.

This example script uses the following classes and functions:

  • parsePeaksMzML.Reader
  • getWindow.PeakLocation
  • getWindow.PeakLocation.getPeaks_mzWindow

Example (See the comments in the source code for detailed information on all the steps.):

>>> from pyMS import parsePeaksMzML
>>> from pyMS import getWindow
>>> peaks = parsePeaksMzML.Reader('example_peak_file.peaks.mzML')                # read in the peaks file and make an instance of parsePeaksMzML.Reader class
>>> peakLocation = getWindow.PeakLocation(peaks)                                 # make an instance of getWindow.PeakLocation using the parsePeaksMzML.Reader instance as input
>>> peakList = peakLocation.getPeaks_mzWindow(300,500)                           # get a list of peaks that have an m/z value between (including) 300-500

massWindow_XIC_plot(retentionList, intensityList, outpath='XIC_plot.png', title='XIC_plot', xlab='retention time (seconds)', ylab='summed intensity', extra_arguments={'type': 'l'})

source code 

Plot XIC for a mass window (aggregated) against run time.

Essentially you are taking the area under the intensity-m/z curve as the single intensity point.

Parameters:
  • retentionList (list) - Sorted list of the retention times corresponding to the intensities in intensityList.
  • intensityList (list) - Sorted list of intensities correpsonding to the retention times in retentionList
  • outpath (string) - The path and name of the file to write out (default = XIC_plot.png)
  • title (string) - Title for the plot (default = XIC_plot)
  • xlab (string) - Description of the x-axis of the plot (default = retention time (seconds))
  • ylab (string) - Description of the y-axis of the plot (default = retention time (seconds))
  • extra_arguments (dict) - extra arguments to give to the plot, with as key the argument name and value the argument value (can use all the R arguments [see http://stat.ethz.ch/R-manual/R-devel/library/graphics/html/plot.html]). (default = {'type':'l'})
Raises:
  • TypeError - intensityDict or extraArguments not a dict and/or outpath, title, xlab, ylab not a string

    {Example:}

    Plot XIC plot with m/z between 450-452

    >>> import pymzml
>>> import mzmlFunctions
>>> mzmlInstance = pymzml.run.Reader('example_file.mzML')
>>> intensityFromSpectra = mzmlFunctions.IntensityFromSpectra(mzmlInstance)
>>> retentionList, intensityList = intensityFromSpectra.getIntensityFromMZwindow(450, 452)
>>> massWindow_XIC_plot(retentionList, intensityList)

parent_to_XIC_plot(mzmlInstance, parentMass, maxCharge=4, outpath='XIC_plot.png', title='XIC_plot', xlab='retention time (seconds)', ylab='summed intensity', extra_arguments={'type': 'l'})

source code 

Makes an XIC overlapping plot of all the charge variations of the parentving the sum and RT as the y and X point respectively for the plot. mass, up to the charge given to maxCharge. E.g., if you run parent_to_XIC_plot(1200, 4), this will plot the XIC for 600 (2+), 400 (3+) and 300 (4+).

Parameters:
  • mzmlInstance (pymzml.run.Reader) - Instance of pymzml.run.Reader class which you want to use to plot the XIC from
  • parentMass (Number) - The mass of the peptide you want the different charge plots to be made from.
  • outpath (string) - The path and name of the file to write out (default = XIC_plot.png)
  • title (string) - Title for the plot (default = XIC_plot)
  • xlab (string) - Description of the x-axis of the plot (default = retention time (seconds))
  • ylab (string) - Description of the y-axis of the plot (default = retention time (seconds))
  • extra_arguments (dict) - extra arguments to give to the plot, with as key the argument name and value the argument value (can use all the R arguments [see http://stat.ethz.ch/R-manual/R-devel/library/graphics/html/plot.html]). (default = {'type':'l'})
  • maxCharge (Number) - The maximum z charge to calculate the m/z for. (default=4)
Raises:
  • TypeError - intensityDict or extraArguments not of tpye dict and/or outpath, title, xlab, ylab not a string and/or parentMass and maxCharge not of type int and/or mzmlInstance not of type pymzml.run.Reader
  • RuntimeError - maxCharge has to be 2 or higher

    Example:

    Plot all peptides with charge 2-4 for mother peptide weight 1500

    >>> import pymzml
>>> mzmlInstance = pymzml.run.Reader('example_file.mzML')
>>> parent_to_XIC_plot(mzmlInstance, 1500)                    #runs with default charge 2-4, outputs to XIC_plot

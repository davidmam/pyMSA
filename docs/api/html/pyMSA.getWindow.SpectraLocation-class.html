getSpectra_mzWindow(self, mzStart, mzEnd)

source code 

Get all the spectra of which the mz value is within mzStart and mzEnd.

Parameters:
  • mzStart (float) - The lowest mz value of the window to retrieve spectra from
  • mzEnd (float) - The highest mz value of the window to retrieve spectra from
Returns: list
A list of all the spectra that have an mz between mzStart and mzEnd
Raises:
  • TypeError - rtStart or rtEnd is not of type int

    Example:

    Print a list of the spectra that have an m/z between 400 and 500:

    >>> import pymzml
>>> spectra = pymzml.run.Reader('example_featureXML_file.featureXML')
>>> spectrumLocation = getWindow.SpectraLocation(spectra)
>>> for spectrum in spectrumLocation.getSpectra_mzWindow(400, 500):
...    print spectrum['id']

getSpectra_rtWindow(self, rtStart, rtEnd)

source code 

Get all the features of which the retention time value is within rtStart and rtEnd. The retention time is in seconds (in the .mzml file it stands as minutes)

Parameters:
  • rtStart (float) - The lowest number of the window to retrieve spectra from
  • rtEnd (float) - The highest number of the window to retrieve spectra from
Returns: list
A list of all the spectra that have an mz between mzStart and mzEnd
Raises:
  • TypeError - rtStart or rtEnd is not of type int

    Example:

    Print a list of the spectra that have an retention time (in seconds) between 400 and 500:

    >>> import pymzml
>>> spectra = pymzml.run.Reader('example_featureXML_file.featureXML')
>>> spectrumLocation = getWindow.SpectraLocation(spectra)
>>> for spectrum in spectrumLocation.getSpectra_rtWindow(400, 500):
...    print spectrum['id']
19

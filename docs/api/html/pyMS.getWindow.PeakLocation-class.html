getPeaks_mzWindow(self, mzStart, mzEnd)

source code 

Get all the features of which the mz value is within mzStart and mzEnd.

Parameters:
  • mzStart (float) - The lowest number of the window to retrieve features from
  • mzEnd (float) - The highest number of the window to retrieve features from
Returns: list
A list of all the features that have an mz between mzStart and mzEnd
Raises:
  • TypeError - rtStart or rtEnd is not of type int

    Example:

    Print a list of the features that have an m/z between 499.9 and 500:

    >>> peaks = parsePeaksMzML.Reader('example_peak_file.peaks.mzML')
>>> peakLocation = getWindow.PeakLocation(peaks)
>>> for peak in peakLocation.getPeaks_mzWindow(350,500):
...     print peak

getPeaks_rtWindow(self, rtStart, rtEnd)

source code 

Get all the features of which the mz value is within mzStart and mzEnd. NOTE: The rt is in seconds !!!

Parameters:
  • rtStart (float) - The lowest number of the window to retrieve features from
  • rtEnd (float) - The highest number of the window to retrieve features from
Returns: list
A list of all the features that have an rt between rtStart and rtEnd
Raises:
  • TypeError - rtStart or rtEnd is not of type int

    Example:

    Print a list of the features that have a retention time between 450 and 500 seconds

    >>> peaks = parsePeaksMzML.Reader('example_peak_file.peaks.mzML')
>>> peakLocation = getWindow.PeakLocation(peaks)

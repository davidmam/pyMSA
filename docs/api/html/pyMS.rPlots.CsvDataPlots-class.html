getParams(self, kwargsDict)

source code 

Has a dictionary with default values which can be overriden by arguments given to kwargsDict. The default dictionary looks like this: The defaultdict looks like this (and these key value pairs can be given to kwargsDict): defaultDict = {'title':self.title, 'xlab':self.xlab, 'ylab':self.ylab, 'width':self.pngWidth, 'height':self.pngHeight,'logBase':self.log,'vector1Name':'vector 1', 'vector2Name':'vector2'}

Parameters:
  • kwargsDict (dictionary) - A dictionary containing all the extra input given to a function, passed to **kwargs
Returns: dictionary
The keys and values of the default dictionary, only the values overriden that were given to kwargsDict
Raises:
  • TypeError - kwargsDict not of type dict

    Example:

    Overriding some default values and leaving some as they are:

    >>> def exampleFunction(**kwargs):
...    csvDataPlots = CsvDataPlots()
...    return csvDataPlots.getParams(dict(kwargs))     # have to make a dictionary out of kwargs
>>> print exampleFunction(title='example getParams', xlab = 'example 2 kwargs') # only override tile and xlab, rest stay default
{'vector1Name': 'vector 1', 'xlab': 'example 2 kwargs', 'logBase': 10, 'title': 'example getParams', 'ylab': 'y', 'vector2Name': 'vector2', 'width': 300, 'height': 300}

  • outPath (string) - Path for the output file
